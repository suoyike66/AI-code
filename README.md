AI Code Generator Platform - 项目介绍
🎯 项目概述
AI Code Generator Platform 是一个基于人工智能的智能代码生成平台，用户只需通过自然语言描述需求，系统即可自动生成高质量的前端代码。项目采用 Spring Boot 3.5.2 + Java 21 + LangChain4j + LangGraph4j 技术栈，实现了从需求分析、代码生成、质量检测到自动化部署的完整闭环。
 
✨ 核心亮点
1️⃣ 智能化代码生成
🎯 智能路由：根据需求复杂度自动选择最优生成策略（HTML / 多文件 / Vue 项目）
⚡ 流式响应：SSE 实时推送生成进度，用户可即时看到代码生成过程
🔄 工作流编排：基于 LangGraph4j 实现 6 节点协作流程（图片收集 → Prompt 增强 → 路由 → 代码生成 → 质检 → 构建）
2️⃣ 质量保障机制
✅ AI 质检回环：自动生成代码后进行质量检测，失败则自动重试修复
🛠️ 工具调用：支持文件读写、修改、删除等操作，AI 可自主调整代码
📊 多维度检查：语法正确性、文件完整性、依赖关系验证
3️⃣ 自动化部署
🚀 一键部署：生成 6 位随机 deployKey，应用立即可访问
🏗️ Vue 项目构建：自动执行 npm install + npm run build
📸 自动截图：部署后使用 Selenium 截取应用封面并上传腾讯云 COS
4️⃣ 企业级特性
🔒 限流保护：基于 Redis 滑动窗口算法，每用户每分钟 5 次请求限制
👤 权限控制：RBAC 角色管理，支持普通用户和管理员
📈 实时监控：集成 Prometheus + Actuator，监控 JVM、HTTP、AI 调用等指标
💾 多级缓存：Caffeine 本地缓存 + Redis 分布式缓存
